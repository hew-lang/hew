# Sandbox VM golden fixtures

Each fixture directory contains:

- `main.hew` — the learner-visible Hew source.
- `expected.trace.json` — the deterministic Trace Schema v0 contract.
- `bytecode.json` — generated by `make sandbox-fixtures`, unless the fixture is
  listed in the enforced deferred block below.

The fixture catalog is declared in [`manifest.json`](manifest.json) and described
in [`../specs/golden-fixtures-v0.md`](../specs/golden-fixtures-v0.md).

Regenerate compiled fixture bytecode from the repository root:

```sh
make sandbox-fixtures
```

Check mode is the CI gate:

```sh
make sandbox-fixtures-check
```

Fixtures that currently exercise diagnostics only (`09-compile-type-error` and
`10-sandbox-reject-filesystem`) intentionally have no bytecode artifact. Fixtures
below are deferred because the checked-in source is ahead of the canonical
sandbox bytecode emitter or, for fixture 30, no source fixture exists yet. The
