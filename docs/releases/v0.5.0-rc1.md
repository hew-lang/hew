# Hew v0.5.0-rc1

Release candidate for v0.5.0. These artifacts exist for pre-release
verification across platforms — checksums, packaging layout, `hew --version`,
and compile-and-run smoke coverage on macOS (arm64), Linux (x86_64 and
aarch64), and Windows (x86_64).

This is not a stable release. Install the final v0.5.0 release once it is
published; the full release notes accompany that tag.
